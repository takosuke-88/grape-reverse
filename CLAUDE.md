# プロジェクトの基本ルール
- 新しいページ（例：`src/pages/` 配下のファイル）を新規作成する際は、必ずファイル上部で `src/components/Seo.tsx` をインポートすること。
- ページコンポーネント内に `<Seo />` を配置し、そのページに最適な `pageTitle`、`pageDescription`、`pagePath` をPropsとして必ず渡すこと。
- SEOの設定漏れは重大なエラーとみなすため、ユーザーからの指示がなくとも自発的に実装に含めること。
- 【重要】既存ページの改修や本文の修正を行う際、ユーザーから明示的な指示がない限り、すでに設定されている `<Seo />` の Props（特に `pageDescription`）は勝手に変更せず、元の内容を維持すること。
