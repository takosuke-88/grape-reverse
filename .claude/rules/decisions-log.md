# Decisions Log (意思決定・変更履歴)

このファイルは、過去に行われた重要な仕様変更、機能の追加・削除の歴史を記録し、AIが古い仕様にコードを「先祖返り」させるのを防ぐためのログです。実装時に過去の設計を覆す指示がない限り、ここに記載された決定を厳守してください。

## 2026-05-23: BIG/REGカウンターのホールド機能の廃止（差し戻し）
- **決定事項**: 一度検証導入した「0.5秒長押し（ホールド）での直接入力（テンキー）起動およびGlow/ゲージ演出」を**全面的に廃止**。
- **理由**: 判定エリアの暴発や、ホール実戦中における操作感の違和感、直感性の低さを解消するため。
- **現行仕様**: すべてのカウンター（ゲーム数、小役、ボーナス）において、**中央数字部分（左側30%領域）をワンタップした瞬間に直接入力キーパッドが一発で起動する仕様**に完全回帰・統一。

## 2026-05-23: ヘッダーブランド表記の変更と2行自動改行・Stickyの復元
- **決定事項**: 最上部ヘッダー左側のテキストを `grape-reverse` から大文字の `Grape Reverse` に更新。画面が狭くなった際、右側メニューが下に回り込んでStickyの高さを破壊するバグを修正。
- **現行仕様**: 右側メニューは1行を維持、左側のロゴ自体が縦2行（1行目: `Grape`、2行目: `Reverse`）に改行される。スクロール時は、ロゴバーは画面外に消え、「機種名バー（マゼンタ）」と「ナビゲーションバー（グレー）」の2行だけがセットで最上部に固定される。

## 2026-05-24: ぶどう逆算の機種別Config注入（多機種対応設計）
- **決定事項**: BIG/REGの払出枚数・ぶどう払出・ベース補正係数を `MachineConfig` に持たせ、計算式は `config.*` を参照して動かす。定数のハードコード（マイジャグ固定）は**禁止**。
- **理由**: `MachinePageFactory` による設定注入アーキテクチャと統一し、ハナハナ等の追加時にロジックを書き換えずConfig追加だけで対応するため。
- **現行仕様**: `specs.payouts` / `specs.reverseCalcProbDenominators` を `machine-schema.ts` で定義。実装詳細は `architecture.md` §5 を参照。

## 2026-05-24: ベース補正係数（baseCorrectionFactor）の採用（設計メモ・未実装）
- **決定事項（当時）**: 補正なし（A）は却下。Configに「1Gあたりのぶどう・ボーナス以外の小役払出期待値」を定義し、総ゲーム数を掛けて差し引く方式（B+Cハイブリッド）を採用する方針とした。

- **参照値（将来用メモのみ）**: マイジャグV **0.494**、キングハナハナ **0.526** / ベル・ぶどう払出 **10**。

## 2026-05-24: コラム記事の管理方針

## 2026-06-01: ぶどう/ベル逆算ロジックの確定（`GrapeReversePage.tsx`）
- **決定事項**: 差枚は **総払出 = 総投入 + 差枚**（`coinIn + diffCoins`）。リプレイ補正 `(G/replayDenom)×3` は**チェリー狙い・フリー打ちの両方**で同一控除。2モードの差は**チェリーこぼしのみ**（フリー打ちは `cherryFreePlayRate`、未指定時 **2/3**）。払出は `payouts.grape ?? payouts.bell`、チェリーは `payouts.cherry`（ハナ4/ジャグラー2）。ハナで分母未設定時チェリー **1/36**。
- **理由**: 差枚符号逆転（`coinIn - diff`）やリプレイをフリー打ちだけ引く誤りでベル1/10台表示になるバグを解消。21コマ系のフリー打ちチェリー理論取得率 2/3 を反映。
- **現行仕様**: 設定推定入力は**チェリー狙い**の推計回数。差枚入力は **0 以上のみ**（マイナス差枚未対応）。逆算ページの localStorage キーは `grape-reverse-data-grape-mode-${machineId}`（小役カウンター本体とは別キー）。

## 2026-06-01: ハナハナ「ベル回数」カウンター配色（アンバー）
- **決定事項**: `getElementTheme` で `id === "bell-count"` のみ **アンバー**（`#f59e0b`）。`id.includes("grape")` はジャグラーぶどうとして**緑のまま**。ランプ要素（`-white` / `-blue` 等）は従来どおり suffix 優先。

- **現行仕様**: 明るい背景ランプ（`-white`）は `textColor: #1e293b` で視認性確保。
